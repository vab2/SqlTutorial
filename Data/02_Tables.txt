Creating Your Own Tables
========================

CREATE TABLE tablename
  (
    column_name data_type,
    column_name data_type
  );
  
CREATE TABLE cats
  (
    name VARCHAR(100),
    age INT
  );
  
-------------------------------------------------------------------------------

Show Tables
===========

SHOW TABLES;
 
SHOW COLUMNS FROM tablename;
DESC tablename;

-------------------------------------------------------------------------------

Dropping Tables
===============

Dropping Tables
DROP TABLE <tablename>; 

A specific example:

DROP TABLE cats; 

Be careful with this command!
